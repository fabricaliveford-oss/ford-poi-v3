# FORD POI v3.0

Plataforma web para gerenciar Pontos de Interesse (POIs) em 12 regionais Ford.

## ⚡ Quick Start

1. Atualize `config.js` com sua URL de deployment
2. Abra `index.html` no navegador
3. Pronto!

## 📚 Documentação

- `docs/DEPLOYMENT.md` - Guia completo de deployment
- `src/backend.gs` - Código do Apps Script
